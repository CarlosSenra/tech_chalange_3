# Projeto Fiap Tech Chalenge 3

O objetivo do projeto é explorar dados de voos de empresas aereas e criar alguns modelos supervisionados e não supervisionados, com esses dados.

### Dados para o projeto

São utilizados três csv's:
    - airlines.csv
    - airports.csv
    - flights.csv

Turante o projeto é feito um breve tratamento nos dados para melhorar a análise e o treinamento dos modelos. O tratamento ele foi sendo feito durante as primeiras utilizações dos dados, enquanto eram feitas as analises exploratórias de dados, em que o resultado final estão nos notebooks de EDA, de forma organizada.
